# Computational_Modeling_of_Cognition_and_Behavior
SusTech BME Machine Learning &amp; NeuroEngineering 2021

本仓库是由深圳 Southern University of Science and Technology（SUSTech）生物医学工程系“机器学习与神经工程”的资源合集。涵盖了“认知与行为的计算建模”（Computational Modeling of Cognition and Behavior）相关的讲义、教材资料、Python／R 实践教程等，旨在帮助学生理解并运用计算模型来研究人类认知及行为机制。
